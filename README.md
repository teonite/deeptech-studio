## Install deps

Make sure you have recommended version of Node.js -> [.nvmrc](.nvmrc)

Install dependencies with [pnpm](https://pnpm.io/)

```bash
nvm install
nvm use
npm i -g pnpm
```

```bash
pnpm install
```

## Launch dev server

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
